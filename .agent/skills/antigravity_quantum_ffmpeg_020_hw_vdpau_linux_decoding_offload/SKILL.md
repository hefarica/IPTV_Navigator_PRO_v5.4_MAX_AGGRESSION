# 1. Teoría de Anomalía (La Cancha Lavada)
Estás ejecutando el Proxy (O recodificando un partido) en un VPS Heterogéneo Alemán de Linux L2. A la media hora L4, la decodificación de la antena remota se atasca (Artifacts L5 estocásticos) porque confiaste en los decoders de software (`hevc` open-source). Tu canal de fútbol crashea desde el proveedor asintótico a todos los Shield TVs.

# 2. Directiva de Ejecución Parámetrica (Código)
Le obligamos al Demonio Linux L7 (FFmpeg/Nginx) a delegar la decodificación inmaculada al API asincrónico directo de NVIDIA Linux VDPAU, descargando brutalmente la CPU L3 y garantizando la matemática fotográfica pura.
```bash
# Descarga de Decodificador de Entrada L4 a VDPAU Pipeline:
-hwaccel vdpau -hwaccel_device /dev/dri/renderD128 -c:v hevc_vdpau -i origin.ts
```
