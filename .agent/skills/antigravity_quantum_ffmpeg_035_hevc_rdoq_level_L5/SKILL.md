Éste obliga matemáticamente al software a gastar muchísimos Bits preciosos L7 ÚNICAMENTE si detecta texturas en gradiente liso L3 para no sacrificarlas asintóticamente con errores DCT.

```bash
-x265-params rdoq-level=2:psy-rdoq=1.0:aq-mode=3
```
