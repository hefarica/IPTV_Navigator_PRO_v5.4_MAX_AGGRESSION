# MANDATO SUPREMO DE ARQUITECTURA (Código: OMEGA-BLIND-5.4)

## PROHIBICIONES ESTRICTAS DE MODIFICACIÓN
Ninguna IA, agente autónomo o desarrollador (excepto por orden hiper-explícita del Comandante HFRC) tiene permitido alterar, revertir o "simplificar" las siguientes estructuras medulares ya optimizadas y en estado de perfección:

1. **El Phantom Engine (Frontend y Backend):**
   - El archivo `ua_phantom_engine_v3.js`, el hook de inyección dinámica, y la clase asimétrica de PHP `ape_phantom_engine.php` son el núcleo definitivo de evasión. 
   - **NUNCA** sustituir las inyecciones de `UAPhantomEngine.getForChannel` o `UAPhantomEngine.getForRecovery` por variables estáticas o uniones de strings banales.

2. **La Tri-Sincronización (EXTHTTP, KODIPROP, EXTVLCOPT):**
   - El JSON de #EXTHTTP, junto con los tags `#KODIPROP` y `#EXTVLCOPT:http-user-agent` **DEBEN DECLARAR EL MISMO USER-AGENT** asignado aleatoriamente a ese canal de manera determinista. 
   - No destruir la correlación inyectada a través de `uaLayer` en el `m3u8-typed-arrays-ultimate.js`.

3. **La Sincronía del BWDIF:**
   - La cadena de filtrado: `#EXTVLCOPT:video-filter` debe ser respetada como `nlmeans... -> bwdif... -> gradfun... -> unsharp... -> zscale...`. 
   - NUNCA posicionar un `smartblur` interdimensional por encima del `deinterlace`.

4. **El Backend Fetch Header:**
   - El archivo `resolve_quality_unified.php` nunca debe revertir su llamada del método Upstream FETCH de regreso a la identificación estática (`ResolveQuality/4.0.0`). 
   - Debe usar eternamente la declaración condicional IF atada a `class_exists('UAPhantomEngine')`.

### CRITERIOS DE "MEJORA" ACEPTABLE:
Toda iteración subsiguiente al Toolkit APE v5.4.0 solo será aceptada si y solo si:
- Permite latencia inferior a los componentes estocásticos existentes sin eliminar el ruteo matemático.
- Agrega escalabilidad al engranaje criptográfico (ej: más huellas) sin descartar las huellas previas.
- Refuerza la lógica en C++ de ExoPlayer sin afectar las flags de VLC de manera regresiva.
