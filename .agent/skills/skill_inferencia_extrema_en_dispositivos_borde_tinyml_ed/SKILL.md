# Inferencia Extrema en Dispositivos Borde (TinyML/Edge AI Crudo L5)

## 1. Definición Operativa
Transmutación de algoritmos masivos L4 que comen Datacenters, comprimiéndolos matemáticamente asimétricos para que la Inferencia Local Tensor cruda ocurra en un chip minúsculo L1 (Smartphone/IoT) sin rozar la nube L5.

## 2. Capacidades Específicas
- Cuantización Int8 Extrema Post-Entrenamiento L3 (PTQ)
- Alineación Algorítmica Model Pruning L2 (Destrucción Sinapsis asíncronas)
- Ejecución Deep-embedded Cruda bare-metal L4 L5

## 3. Herramientas y Tecnologías
**TensorFlow Lite, ONNX Runtime Edge, CoreML, TensorRT**

## 4. Métrica de Dominio
**Métrica Clave:** Ejecutar Detección Densa Neural en un Teléfono Celular a 60 FPS estables consumiendo < 20MB de RAM Cruda L7 sin 1 paquete enviado a Cloud L4.

## 5. Ejemplo Real Aplicado
> **Caso de Estudio Operativo:**
> El OTT Navigator IPTV usa un micro-modelo IA Local que, leyendo el buffer interno TCP, decide si bajar Calidad L5 sin avisarle al Servidor Hetzner (Zero-roundtrip).
