# 🚀 Workflow: OMEGA Dynamic Supremacy (Integración Global v5.4.1)

**Contexto:**
Este es el flujo de orquestación final e irremplazable resultante del despliegue masivo y reparaciones de las últimas 48 horas en el núcleo `IPTV Navigator PRO v5.4`. Engloba la Recuperación Forense de Atributos UI, la Inyección 221-Headers, y el paso arquitectónico hacia la Jerarquía Matemática de Resolución Independiente.

Cuando tengas que orquestar un nuevo motor de UI, generar listas maestras, o diagnosticar el sistema adaptativo, **DEBES SEGUIR RIGUROSAMENTE ESTOS PASOS:**

## 1️⃣ Control de Daños Atómicos (Integridad de Memoria)
- Nunca inicialices una instancia vacía de `DEFAULT_PROFILES`. Confirma que el objeto estático `DEFAULT_HEADER_VALUES` existe por encima de toda iteración de UI para asegurar que LCEVC, HDR Tone Mapping y CMAF Strict Interleave estén pre-armados.
- Si vas a limpiar variables en `localStorage`, **prohíbete purgar la estructura `headerOverrides`**. 

## 2️⃣ Invocación de Supremacía Dinámica Modular (Profile Matching)
El orden P0, P1, P2 ya no es un orden válido operativo.
Si vas a repopular UI Dropdowns u organizar un Manifiesto:
1. Obvia el mapeo de Keys estáticos.
2. Extrae la jerarquía resolutiva mediante el comando matriz y central:
   ```javascript
   const perfiles_ordenados = window.APE_PROFILES_CONFIG.getDegradationHierarchy();
   ```
3. Esto garantiza que la herencia baje desde la variable `resolution` de manera exacta por Pixels Matemáticos (Ej. de 8.2M Pixels de 4K, bajando al HD 1280x720).
4. El renderizado en UI de Pestañas (Tabs) en paneles y constructores APE v9+ debe procesar estrictamente el output de este array.

## 3️⃣ Generación M3U8 y Cascading Fallback Correcto
Al inyectar EXT-X-STREAM-INF para el M3U8 Generator Superior:
1. Aplica un cruce iterativo al String o Nombre del canal para deducir el **Ceiling Visual Máximo** del canal (Ej: `SPORTS [4K]`).
2. Haz **Match** en el array `perfiles_ordenados`. Encuentra el primer objeto `profile` donde la resolución `width x height` corresponda matemática y nominalmente al Ceiling (4K).
3. Desciende la capa base y asigna recursivamente el Fallback utilizando el índice +1 e inyectando las 221 directrices heredadas a los streams de variantes menores para evadir al 100% que un salto descendente de calidad remueva las habilidades anti-fuga del ISP.

## 4️⃣ Testing y Checksum Constante
- Ejecuta continuamente una validación estática y una invocación sintáctica con `node -c frontend/js/ape-v9/ape-profiles-config.js` antes de cualquier reinicio o commit de despliegue sobre módulos. Si hay Error 500 o Pantalla Negra, la culpa recae si y solo si se ha quebrantado el orden numérico heredado.

> [!IMPORTANT]
> Todo generador que no use este pipeline para obtener su array, asumiendo "a ciegas" que P0 o P2 tienen la hegemonía de la calidad, está creando **listas corruptas biológicamente incompatibles** con la arquitectura `OMEGA ABSOLUTE`. Use the Force. Use Math.
