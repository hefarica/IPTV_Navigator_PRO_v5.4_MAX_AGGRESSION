# V4.1 (Mitigaciones inmediatas) — IPTV Navigator PRO v3.2.1

## Qué se agregó
- UI en `index-v4.html`: Auto-Detect (V4.1), selector de Player, toggle Serverless (V5.0).
- Módulos nuevos:
  - `js/v41/headers-intelligent-v3-adapter-v41.js`
  - `js/v41/conflict-detector-v41.js`
  - `js/v41/telemetry-feedback-v41.js`
- Integración:
  - `app.js` ahora pasa `antiFreezeLevel`, `autoDetectLevel`, `targetPlayer`, y opciones V5.
  - `export-module.js` genera headers por canal usando `ULTRA_HEADERS_MATRIX.getAllHeadersForLevel()` y aplica:
    - Auto-detect (si está activo)
    - Sanitización por player (conflictos)
    - Telemetría pasiva en localStorage

## Cómo usar (frontend)
1. Activa/desactiva **🤖 Auto-Detect Nivel (V4.1)**.
2. Elige **🎯 Player** (Genérico/VLC/Kodi/OTT/TiviMate/Smarters/Perfect Player).
3. Genera tu lista normalmente.

## Notas
- La telemetría es local (solo browser).
- Serverless (V5) es opcional y requiere desplegar el Worker incluido en el ZIP V5.
