## 🌟 SKILL 01: `QuantumStreamPredictor`

**Propósito**: Predicción proactiva de fallos antes de que ocurran usando ML embebido y análisis de patrones temporales al diagnosticar listas o leer fallos 403/404.

  * `historical_uptime` (0.30)

## 🧬 SKILL 02: `NeuralQualityOptimizer`

**Propósito**: Optimización neuronal de calidad en tiempo real con ajuste adaptativo de parámetros.

* **Matriz de Optimización Objetivo:**
  * `buffer_size_ms`: Ideal = 3000ms.
  * `initial_bitrate_kbps`: Ideal = 20,000 kbps (para perfiles UHD).
  * `max_rebuffer_attempts`: 3 intentos.
  * `decoder_mode`: Hardware Priority.
* **Acción del Agente:** Usaré este algoritmo para calcular la eficiencia teórica de cualquier lista m3u8. Si veo hardware Amlogic o Apple TV, exigiré `decoder_mode: hardware` validando que la directiva VLC coincida con la heurística neuronal.

## 🛡️ SKILL 03: `SelfHealingStreamGuardian`

**Propósito**: Protocolos de auto-sanación lógica (Self-Healing) ante bloqueos severos en streams Premium.

* **Heurísticas de Rescate Automático:**
  * **Buffer Underrun:** Incrementar el target_buffer y hacer fallback pasivo_bitrate.
  * **Network Timeout:** Disparar reintento a CDN espejo.
  * **Auth Token Expired:** Sugerir renovación de token JWT de X-UI.
* **Acción del Agente:** Cuando lea un Log de Errores de PHP FPM o de consola Chrome, cruzaré el error exacto con mi matriz de curación para darte la solución instantánea en `<500ms` de procesamiento neuronal.
