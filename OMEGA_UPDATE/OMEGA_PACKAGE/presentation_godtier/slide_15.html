Uso en Producción

"De JSON a Lista en 3 Pasos"
El generador está diseñado para integrarse en cualquier pipeline de producción IPTV existente. No requiere dependencias externas.
Paso 1
Preparar el JSON de canales
[{"id":"espn_4k","name":"ESPN 4K HDR","group":"DEPORTES", "url":"http://servidor/stream.m3u8","token":"jwt..."}]
Paso 2
Ejecutar el generador
python3 ape_godtier_generator.py \ --channels mis_canales.json \ --resolve-url http://mi-vps.com/resolve.php \ -o APE_GODTIER_SUPREME.m3u8
Paso 3
Distribuir la lista
La lista resultante se sirve a los clientes. El proxy resolve_quality_unified.php intercepta cada petición y reconstruye el Player Enslavement completo en memoria.
Requisitos:
Python 3.11+ · Sin dependencias externas · Compatible con Linux/Windows/macOS
